General information: Application of Autoencoders in image denoising. An autoencoder contains an encoder and decoder. These two parts function automatically and give rise to the name “autoencoder”. In denoising, image/data is corrupted through the addition of random noise, and the model is trained to predict the original uncorrupted data. Denoising autoencoders can be used for aspects such as cleaning old stained scanned images or feature selection in projects such as cancer biology.

Now, we need to add random noise to the image to corrupt the images:

   ## example
   noise_factor = 0.5
   x_train_noisy = x_train + noise_factor * np.random.normal(loc=0.0, scale=1.0, size=x_train.shape)
   x_test_noisy = x_test + noise_factor * np.random.normal(loc=0.0, scale=1.0, size=x_test.shape)

Then define a simple Network architecture including two convolutional layers x = layers.Conv2D(32, (3, 3), activation=‘relu’, padding=‘same’)(input_img) x = layers.MaxPooling2D((2, 2), padding=‘same’)(x) x = layers.Conv2D(32, (3, 3), activation=‘relu’, padding=‘same’)(x) encoded = layers.MaxPooling2D((2, 2), padding=‘same’)(x)

  # At this point the representation is (7, 7, 32)

  x = layers.Conv2D(32, (3, 3), activation='relu', padding='same')(encoded)
  x = layers.UpSampling2D((2, 2))(x)
  x = layers.Conv2D(32, (3, 3), activation='relu', padding='same')(x)
  x = layers.UpSampling2D((2, 2))(x)
  decoded = layers.Conv2D(1, (3, 3), activation='sigmoid', padding='same')(x)
  autoencoder = keras.Model(input_img, decoded)
  autoencoder.compile(optimizer='adam', loss='binary_crossentropy')

  • Conclusion

The denoising autoencoder manages clearing the corrupted image without overly smoothing the image as it is done by non-AI image clearing approaches such as traditional filtering! It should be noted that convolutional neural network (CNN) was considered because of its effectiveness in denoising and spatial relations preservation within the image. This generated network may be used for future projects relevant to audio denoising.
